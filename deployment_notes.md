# Streamlit Community Cloud Deployment Requirements

Based on the documentation, to deploy to Streamlit Community Cloud:

## Requirements:
1. **GitHub Repository**: Code must be in a GitHub repository (public or private)
2. **File Structure**: 
   - Main app file (e.g., `app.py`)
   - `requirements.txt` for Python dependencies
3. **Streamlit Community Cloud Account**: Free account connected to GitHub
4. **Deployment Process**: Click "Create app" in workspace, fill in app info, click "Deploy"

## Key Points:
- Free tier allows unlimited apps if source code is publicly available
- Apps deploy in just a few minutes
- No server management required
- Automatic updates when you push to GitHub

## Steps to Deploy:
1. Create GitHub repository with the app code
2. Add requirements.txt file
3. Sign up for Streamlit Community Cloud
4. Connect GitHub account
5. Deploy from the dashboard
